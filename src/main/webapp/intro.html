성적 관리 시스템



기능 사용 방법
전체 학생 조회 조회 　학생 목록 전체 조회
성적 관리 　선택 학생의 성적 목록 전체 조회. 조회 후, 성적 추가/수정/삭제 가능.
수정 　학생 정보 수정. 이름만 수정 가능, 학번 수정 불가.
삭제 　학생 정보 삭제. 학생 정보 삭제 시, 연관 성적 데이터 일괄 삭제.
학생 검색 조회 　학생 이름, 학번, 혹은 이름과 학번을 지정하여 검색
성적 관리 　선택 학생의 성적 목록 전체 조회. 조회 후, 성적 추가/수정/삭제 가능.
수정 　학생 정보 수정. 이름만 수정 가능, 학번 수정 불가.
삭제 　학생 정보 삭제. 학생 정보 삭제 시, 연관 성적 데이터 일괄 삭제.
학생 추가 추가 　학생 정보 추가. 이름 정보 입력 시, 학번 자동 부여.
